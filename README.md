# America_Through_The_Journalist_Glass
By LuisPablo Padres
Topic: The distribution of information through journalistic publications throughout Revolutionary America. 
Data Source: Revolutionary Crossroads Chronicling America
This project follows a database of journalistic issues from a variety of news publications from multiple states in Revolutionary America. The purpose of the project is to view this time in our country thorugh a journalisitc perspective. What states were pushing out the most information? What was the information? What years were the most information codensed? All these questions are meant to push the user into an inmersive experience where they can view the real-time information that was being shared at the time.
